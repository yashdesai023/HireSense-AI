🔄 Data Flow Architecture

  • 🎨 React Frontend: Handles user interactions and displays data
  • ⚙️ Spring Boot Backend: Orchestrates all business logic and API endpoints
  • 🤖 Python ML Service: Processes AI/ML tasks for assessments and resume scoring
  • 💾 MySQL Database: Stores all application data and user information

🔧 Core Module Functions:

  • 🔐 Auth Module: User authentication, authorization, and session management
  • 📝 Job Posting Module: Create, manage, and display job listings
  • 🧠 Assessment Engine: Generate and evaluate skill tests and interviews
  • 📊 Resume Scoring Engine: Analyze resumes and provide skill assessments
  • 📅 Interview Scheduling: Coordinate interview sessions and availability
  • 📈 Analytics Dashboard: Provide insights and reporting for all activities

🔄 Key Interactions:

  • User Requests → Backend: All user actions flow through Spring Boot
  • Backend → Modules: Business logic distributed across specialized modules
  • Modules → Database: Persistent storage for all application data
  • Backend/Modules → ML Service: AI processing for assessments and scoring
  • ML Service → Modules: AI results integrated back into core functionality
